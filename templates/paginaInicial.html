Este projeto foi desenvolvido como parte da disciplina de Desenvolvimento Web, com o objetivo de criar uma plataforma prática e funcional para a administração do cinema do Instituto Federal. O sistema permite organizar sessões de filmes, gerenciar horários, cadastrar produções, além de oferecer aos usuários informações atualizadas sobre a programação disponível. Nosso foco é unir tecnologia e cultura, proporcionando uma experiência acessível tanto para os gestores quanto para o público que frequenta o cinema.
